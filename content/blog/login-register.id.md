### Persiapan
- Download file untuk server-nya disini https://bit.ly/2w6zoXv kemudian,-
- Import database ke local/online server.
- Copy/Cut kode php ke directory _htdocs_ (jika menggunakan xampp). 
- Ubah konfigurasi `connect.php`, sesuaikan dengan setingan phpmyadmin anda contoh : 
`$conn = mysqli_connect('localhost', 'username', 'your_password', 'database_name');`
